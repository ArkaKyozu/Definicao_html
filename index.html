Pagina web

Voltar ao inicio

A pagina web é estruturada atraves do HTML, que é uma Linguagem de Marcação de HiperTexto, criada atraves de tags. O html possue diversas tags, que são palavras chaves da qual possuem diversas funções e caracteristica

Tag HTML

Voltar ao inicio

A tag HTML está presente em todas as paginas, é ela quem define onde será o começo e o fim da pagina, portanto a tag deve por obrigação ser aberta no inicio e fechada ao termino da criação da pagina

Tag head

Voltar ao inicio

A palavra head significa cabeça, tudo que estiver dentro dessa tag, não será visivel para o usuario, portanto ela é utilizada para realizar para o carregamento de funcionalidades extras atraves de links e etc. Dentro da tag head temos uma tag chamada title.

Tag title

Voltar ao inicio

Como dito anteriormente, a tag title fica dentro da tag head, é ela quem define o texto que vem escrito na aba do navegador

Tag body

Voltar ao inicio

Como o proprio nome sugere, a tag body é o corpo do site, tudo que é escrito dentro dessa tag é visivel ao usuario, é nessa tag por exemplo que estou escrevendo esse texto.

tags h1 a h6

Voltar ao inicio

Essas tags são responsaveis por definir o tamanho de titulos e subtitulos, sendo h1 o maior e h6 o menor

tag P

Voltar ao inicio

A tag P é o paragrafo do texto, é aberta antes de iniciar um texto e fechada ao fim de um paragrafo

Manipulação de listas

Voltar ao inicio

O html possue tags para a Manipulação de listas que são muito uteis no dia a dia, para esse tipo de Manipulação temos duas tags OL, UL;

OL: significa lista ordenada, ou seja, sempre que utilizada vai ordenar a lista precedido de numerais

UL: significa lista não ordenada, ou seja, sempre que utilizada vai ordenar a lista precedido por padrão de bolinhas

Alem disso, indifente da tag de Manipulação de lista é preciso utilizar a tag LI, entre elas. A tag li significa item da lista é nela que vai o conteudo da lista

Tag de destaque de texto

Voltar ao inicio

A tag strong: serve para deixar o texto ou palavra em negrito, ou seja, tudo que estiver dentro dessa ta ficará em negrito

A tag i: serve para formatar o texto ou palavra em italico

A tag mark: serve como marca texto, ela marca uma palavra ou texto ao qual se deseja dar um destaque maior

A tag small: todo texto escrito dentro dessa tag vai ficar pequeno

A tag blockquote: serve para criar uma citação na pagina

Tag img

Voltar ao inicio

Essa tag é utilizada para inserir imagens dentro da pagina web, por meio dela podemos inserir imaens locais e da internet

Tag input

Voltar ao inicio

Essa tag insere um campo de interação com o usuario, onde o mesmo pode inserir texto, cores, data, numeros, etc.

Atributos das tags

Voltar ao inicio

No HTML as tags pode ou não possuir Atributos, que são como se fosse caracteristicaque auxiliam na identificação e estilização do conteudo que as envolve, existem atributos gerais e especificos dentro os diversos atributos posso citar: ID, Style, Class, type, entre outros...
